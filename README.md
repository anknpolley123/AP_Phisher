## 🛠️ Installation & Usage

### Prerequisites
Make sure you have `git`, `php`, and `curl` are installed properly.

### Setup
```bash
# Clone the repository
git clone https://github.com/anknpolley123/AP_Phisher.git

# Enter the directory
cd AP_Phisher

# Give execution permission
chmod +x *

# Run the setup script
./setup.sh

# Usage
./AP_Phisher.sh or bash AP_Phisher.sh
